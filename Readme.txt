## General info
Converts a japanese written .pdf file into .docx removing the furigana/kana, from the original text.
You can choose between Left (左), and Right (右), or 2: Middle (真ん中), for Text alignment
	
## Technologies
Project is created with:
* iTextSharp.text;
* BitMiracle.Docotic;
* Spire.Doc;
